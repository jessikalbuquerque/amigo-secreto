# Amigo Secreto 🎁

Um aplicativo simples para organizar sorteios de amigo secreto.

## 📌 Funcionalidades  
- Adicionar nomes dos participantes  
- Realizar o sorteio de forma automática  
- Exibir os pares sorteados
